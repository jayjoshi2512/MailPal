## 🎯 Why MailPal?

Cold emailing is powerful, but most tools either feel clunky or cost a fortune. **MailPal** bridges that gap—a sleek, self-hosted platform that turns your Gmail into a professional outreach machine.

<table>
<tr>
<td width="50%">

### 🚀 **Power Features**
- **Smart Personalization** - Dynamic variables auto-fill names, companies, and custom fields
- **Campaign Orchestration** - Schedule, track, and manage multiple campaigns simultaneously
- **Real-Time Analytics** - Live dashboards showing opens, responses, and engagement metrics
- **AI-Powered Writing** - Gemini AI crafts compelling email copy and subject lines

</td>
<td width="50%">

### 💎 **Delightful Experience**
- **Gmail Native** - Send through your actual Gmail account via OAuth 2.0
- **Bulk Import** - Drop CSV files with hundreds of contacts instantly
- **Template Library** - Save and reuse your best-performing emails
- **Dark Mode** - Easy on the eyes during late-night campaign pushes

</tr>

## 🛠️ Built With Modern Tech

MailPal leverages cutting-edge technologies to deliver a fast, reliable, and scalable experience:

| Layer | Technology | Why We Chose It |
|-------|-----------|-----------------|
| **Frontend** | React 19 + Vite | Lightning-fast dev server and optimized production builds |
| **Styling** | Tailwind CSS + Radix UI | Beautiful, accessible components with utility-first CSS |
| **Backend** | Node.js + Express | Proven, scalable architecture with massive ecosystem |
| **Database** | PostgreSQL | Rock-solid relational database for complex queries |
| **Email** | Gmail API OAuth 2.0 | Maximum deliverability through official Google integration |
| **AI** | Google Gemini | State-of-the-art language model for content generation |
| **Charts** | Recharts | Responsive, customizable data visualizations |

## 🚦 Getting Started

Ready to supercharge your cold email game? Here's what you need:

**Prerequisites:**
- Google Cloud account with Gmail API enabled
- PostgreSQL database (14+)
- Node.js runtime (18+)

**In 3 Minutes:**
1. Clone the repo and install dependencies
2. Configure your Gmail OAuth credentials
3. Set up the PostgreSQL database
4. Launch and start sending campaigns

*Detailed setup guide available in the repository documentation.*
